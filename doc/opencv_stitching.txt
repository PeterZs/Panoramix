OpenCV Stiching测试代码特征匹配与相机参数校正部分的算法结构：

stitching_detailed.cpp

1. 寻找 MatchesInfo 类型的 pairwise_matches
	使用 BestOf2NearestMatcher
	1.1 BestOf2NearestMatcher 在图像集合中寻找两两之间可能的匹配
	1.2 因此我们必须肢解 BestOf2NearestMatcher
	1.3 BestOf2NearestMatcher 对两两图片
		1.3.1 使用 (CpuMatcher/GpuMatcher) 中的 knn 方法寻找匹配
			1.3.1.1 FlannBasedMatcher
			1.3.1.2 
		1.3.2 使用 RANSAC 基于匹配计算 pairwise Homography 矩阵
		1.3.3 寻找 pairwise inliers 数目，计算权重
		1.3.4 仅基于 inliers 来计算最后的 Homography 矩阵

2. 确定 cameras 初始角度
	使用 HomographyBasedEstimator 基于上一步得到的 homography
	基于 max spanning tree 进行计算

3. 调整优化 cameras 的精细参数
	使用 BundleAdjusterReproj 或 BundleAdjusterRay
	BundleAdjuster* 定义了基于 pairwise_matches 和 features 的 误差函数、Jacobian矩阵来进行优化
